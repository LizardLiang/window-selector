# Window Selector - 快速參考

## ⌨️ 快捷鍵總覽

| 功能 | 快捷鍵 | 說明 |
|------|--------|------|
| **全螢幕覆蓋層** | `Ctrl+Alt+Q` | 顯示所有視窗的 DWM 即時縮圖方格 |
| **標籤模式** | `Win+Y` | 在視窗標題列上方顯示字母標籤 |
| 選擇視窗 | `A-Z` | 按對應字母選擇視窗 |
| 確認跳轉 | `Enter` 或 `Space` | 切換到選中的視窗 |
| 取消 | `Esc` | 關閉覆蓋層/標籤模式 |
| 數字標籤 | `Ctrl+1-9` | 為選中的視窗加上數字標籤 |
| 快速跳轉 | `1-9` | 跳轉到已標籤的視窗 |

## 🎯 使用場景

### 全螢幕覆蓋層 (Ctrl+Alt+Q)
**適合**: 視窗被遮擋、最小化、或需要預覽內容時

```
Ctrl+Alt+Q → 看到所有視窗縮圖 → 按字母選擇 → Enter 跳轉
```

**特色**:
- ✅ DWM 即時縮圖預覽
- ✅ 快速列表欄顯示所有視窗
- ✅ 支援數字標籤（Ctrl+1-9）
- ✅ 多螢幕支援

### 標籤模式 (Win+Y)
**適合**: 視窗可見時快速切換（類似 FastWindowSwitcher）

```
Win+Y → 看到視窗上方的標籤 → 按字母直接跳轉
```

**特色**:
- ✅ 極簡視覺干擾
- ✅ 最快切換速度
- ✅ 系統強調色 + 光暈效果
- ✅ 不遮擋視窗內容

## 📊 兩種模式比較

| 項目 | Ctrl+Alt+Q (覆蓋層) | Win+Y (標籤) |
|------|----------------|--------------|
| 視覺呈現 | 全螢幕方格 | 小型標籤 |
| 視窗預覽 | ✅ 即時縮圖 | ❌ 無預覽 |
| 視覺干擾 | 中等 | 最小 |
| 切換速度 | 快 | 極快 |
| 最小化視窗 | ✅ 支援 | ❌ 不支援 |
| 數字標籤 | ✅ 支援 | ❌ 不支援 |

## 🎨 字母分配順序

```
a s d f g h j k l    ← Home Row (最常用)
q w e r t            ← 上排左側
y u i o p            ← 上排右側
z x c v b n m        ← 下排
```

**規則**: 最近使用的視窗獲得 **A**，第二個獲得 **S**，以此類推。

## ⚙️ 設定檔

位置: `%APPDATA%\window-selector\config.toml`

```toml
# 全螢幕覆蓋層 (Ctrl+Alt+Q)
hotkey_modifiers = 16387   # MOD_WIN | MOD_NOREPEAT
hotkey_vk = 81             # VK_Q

# 標籤模式 (Win+Y)
label_hotkey_modifiers = 16387   # MOD_WIN | MOD_NOREPEAT
label_hotkey_vk = 89               # VK_Y
```

### 常用 VK 碼

| 按鍵 | VK 碼 | 按鍵 | VK 碼 |
|------|-------|------|-------|
| A | 0x41 | N | 0x4E |
| B | 0x51 | O | 0x4F |
| C | 0x43 | P | 0x50 |
| D | 0x44 | Q | 0x51 |
| E | 0x45 | R | 0x52 |
| F | 0x46 | S | 0x53 |
| G | 0x47 | T | 0x54 |
| H | 0x48 | U | 0x55 |
| I | 0x49 | V | 0x56 |
| J | 0x4A | W | 0x57 |
| K | 0x4B | X | 0x58 |
| L | 0x4C | Y | 0x59 |
| M | 0x4D | Z | 0x5A |

### 修飾鍵

| 修飾鍵 | 數值 |
|--------|------|
| MOD_ALT | 0x0001 |
| MOD_CONTROL | 0x0002 |
| MOD_SHIFT | 0x0004 |
| MOD_WIN | 0x0008 |
| MOD_NOREPEAT | 0x4000 |

**組合範例**:
- `Ctrl+Alt+Q`: `modifiers = 0x0008 | 0x4000 = 16387`, `vk = 0x51`
- `Ctrl+Alt+S`: `modifiers = 0x0002 | 0x0001 | 0x4000 = 16387`, `vk = 0x53`

## 🚀 快速開始

1. **啟動程式**
   ```bash
   .\target\release\window-selector.exe
   ```

2. **測試覆蓋層**
   - 開啟幾個視窗
   - 按 `Ctrl+Alt+Q`
   - 看到縮圖方格
   - 按字母選擇視窗

3. **測試標籤模式**
   - 確保視窗可見
   - 按 `Win+Y`
   - 看到視窗上方的標籤
   - 按字母直接跳轉

## 💡 使用技巧

1. **數字標籤**: 為常用視窗設定數字標籤（Ctrl+Alt+Q → 選擇視窗 → Ctrl+1），之後直接按 `1` 跳轉

2. **快速切換**: 視窗可見時用 `Win+Y`，視窗被遮擋時用 `Ctrl+Alt+Q`

3. **取消操作**: 任何時候按 `Esc` 都可以取消

4. **多螢幕**: 兩種模式都支援多螢幕，標籤會顯示在所有螢幕上
